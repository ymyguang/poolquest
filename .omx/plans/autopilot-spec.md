# PoolQuest Demo Spec

Implement Dragon Pool as a hackathon MVP around Uniswap V4 Hook semantics.

The app must show an immediately usable game console, clear operation order, wallet/network affordances for X Layer, contract metadata, and a local demo mode that works before deployment.

The contract layer must expose QUSD/DRAGON demo assets, a Hook-like quest state machine, and a router that preserves player identity across wrapped AMM-like actions.
